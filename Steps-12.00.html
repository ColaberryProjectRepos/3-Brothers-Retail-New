Creating Calendar and Refresh Table

Create the Calendar table by selecting “New Table” and using the following DAX code.

Calendar =

ADDCOLUMNS (

    CALENDAR ( DATE (2019, 1, 1), DATE (2020, 12, 31) ),

    "Year", YEAR ( [Date] ),

    "MonthNumber", FORMAT ( MONTH ( [Date] ), "00" ),

    "Month Year", FORMAT ( [Date], "MMM YYYY" ),

    "Week Number", "Week-" & FORMAT ( WEEKNUM ( [Date] ), "00" ),

    "Month - Day", MONTH ( [Date] ) & "-" & FORMAT ( [Date], "dd" ),

    "Year Quarter", FORMAT ( [Date], "YYYY" ) & "/Q" & FORMAT ( [Date], "Q" ),

    "MonthYearSort", YEAR ( [Date] ) & FORMAT ( MONTH ( [Date] ), "00" ),

    "WeekDay", FORMAT ( [Date], "dddd" ),

    "WeekDaySort", WEEKDAY([Date])

)

 

For the refresh table create a blank query. Use the following DAX code to pull the current date and time.

= DateTime.LocalNow()

Make sure to change the table and column name to something more appropriate, such as “Refresh Date.”
