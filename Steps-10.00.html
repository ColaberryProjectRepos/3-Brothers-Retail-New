Downloading and Importing Data

Download the data by clicking the download icon above , and make sure to note where you save it. Locate this save file and extract the data. You will do this by right clicking the zip file and selecting “extract all” in the dropdown menu. Select OK in the popup window to create this new folder in the same parent folder as the download. Next, head back over to the PowerBI so we can import this data. Import the data by clicking "Get Data" and selecting “Text/CSV.” Navigate to where you made the extraction. Within the “Archive” folder you will find a file named “SuperStore_Sales_Dataset.” Select this file and click "Transform Data".
