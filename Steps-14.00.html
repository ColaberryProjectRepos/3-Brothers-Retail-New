Additionally, we will need to change the sort by column for 2 of the columns in our Calendar table. We will do this by selecting the column we want to sort, then selecting Properties: Advanced: Sort by Column. For WeekDay we will sort by WeekDaySort. For Month Year we will sort by MonthYearSort.

You should also take this time to make sure your metrics have the correct data type. For example, make sure all of the percentages have the percentage file type, make sure total sales is a currency.
