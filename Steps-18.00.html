Adding Sub-KPIs and Indicators

For many of these Sub-KPIs, we will utilize the button over the card. Currently, cards do not allow the horizontal alignment of text in the version of PowerBI this dashboard was built in. Please note for all except KPI 4, we will be adding a Sub-KPI as well as an indicator.

Adding Sub-KPI 1:


Insert: Buttons: Blank Button

Size and Position: Height = 39, Width = 99, Horizontal = 149, Vertical = 205

Background: Off, Visual Border: Off, Title: Off

Button Style: State= Default, Shape= Rectangle, Rounded corners= 0 px

Text: Text: Conditional Formatting= IBC Sales MoM%, Font= DIN, Font Size= 23, Font Color= #46B1C9, Horizontal alignment= Right, Vertical alignment= Middle

Icon: Off, Fill: Off, Border: Off, Shadow: Off, Glow: Off, Action: Off

 

Insert: Card: Field= Arrow Sales MoM%

Size and Position: Height = 47, Width = 54, Horizontal = 232, Vertical = 205

Background: Off, Visual Border: Off, Title: Off, Category label: Off

Callout value: Size= 23, Color: Conditional Formatting

Color Formatting: Format Style= Rules, What field should we base this on?= Sales MoM%

Rule #1: If value >= Min and < 0 then Color #D64550

Rule #2: If value > 0 and <= Max then Color #73EC84
